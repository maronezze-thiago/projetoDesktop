# Projeto Sistema Para Mercado


> Esse e um projeto simples para demonstrar as operações CRUD simulando o sistema de um mercado.


### Autor

Thiago de oliveira



###  Pré-requisitos

O que precisa para rodar o projeto:

* Java Runtime(JDK): Versao 17
* Driver jdbc
* MySQL Server

### Funcionalidades

- Adicionar produtos para o banco de dados
- Listar os produtos
- Atualizar os produtos já adicionados
- Deletar os produtos existentes
- Exportar os dados para uma planilha excel
- Importar dados de uma planilha e adicionar ao banco
- Possui cadastro de usuarios com senha encriptada



### Tecnologias usadas

- Java
- MySQL
